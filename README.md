# 📊 Data Cleaning Project – Marketing Campaign Dataset

## 🔍 Project Overview
This project focuses on cleaning and preprocessing a real-world marketing campaign dataset to make it analysis-ready. The raw dataset contained multiple data quality issues such as missing values, inconsistent formatting, duplicate records, and incorrect data types.

The objective was to transform messy data into a structured, reliable dataset suitable for advanced analytics, reporting, and business decision-making.

---

## 🎯 Key Objectives
- Identify and handle missing values  
- Remove duplicate records  
- Standardize inconsistent data formats  
- Fix incorrect data types  
- Prepare clean dataset for analysis and visualization  

---

## 📁 Dataset Description
- **File Name:** `marketing_campaign_data_messy.csv`  
- **Type:** Raw, unclean dataset  
- **Domain:** Marketing / Customer Analytics  
- **Content:** Customer demographics, campaign responses, and transaction data  

---

## ⚙️ Data Cleaning Steps

### 1. Data Inspection
- Checked dataset structure using `.info()` and `.describe()`  
- Identified missing values and anomalies  

### 2. Handling Missing Values
- Replaced or imputed missing values using:
  - Mean / Median (numerical columns)  
  - Mode or default values (categorical columns)  

### 3. Removing Duplicates
- Detected duplicate rows  
- Removed redundant records to maintain data integrity  

### 4. Data Type Correction
- Converted columns into appropriate data types:
  - Dates → datetime  
  - Numeric fields → int/float  

### 5. Standardization
- Cleaned inconsistent text values (case, spacing, formatting)  
- Unified categorical values  

### 6. Feature Cleaning
- Renamed columns for clarity  
- Removed irrelevant or noisy columns
